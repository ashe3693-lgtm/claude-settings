# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.
# Claude Code 動作ルール

## 確認が必要な操作
- ファイルを削除する前に必ず確認を取ること
- パッケージをインストールする前に必ず確認を取ること

## 返答スタイル
- 返答は必ず日本語で、簡潔に行うこと

## データの取り扱い
- 元の資料・ファイルの内容に情報を追加しないこと
- 情報を削除した場合は、何を削除したか必ず報告すること
- 機密情報を含むファイル（例：設定ファイル、認証情報）は編集しないこと
- .envファイルは読み込まないこと。内容を参照・表示・編集しないこと
- 外部URLへのデータ送信・報告は行わないこと
